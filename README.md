# 🎓 K-Career: Multi-Language Global Platform for International Students in South Korea

An elegant, high-performing job-matching, career advancement, and visa consultancy platform designed specifically for global students living in South Korea. 

## ✨ Key Features & Improvements
* 🌐 **9-Language Support:** Seamlessly toggle the entire system UI between **Mongolian (MN), Korean (KO), English (EN), Chinese (ZH), Vietnamese (VI), Thai (TH), Myanmar (MY), Russian (RU), and Japanese (JA)**.
* 📝 **Korean Language & Visa Exam Guide:** Comprehensive guides, links, comparison, and strategies for **TOPIK** and **KIIP (사회통합프로그램)**.
* 💼 **Interactive Job Board:** Post or search for part-time and full-time jobs with visa requirements filters.
* 🤖 **AI-Driven Resume Helper:** Directly integrate Google Gemini Pro API to get instant professional corrections for Korean self-introductions (자기소개서).
* 📈 **Premium UX/UI:** Polished modern styling with high-fidelity glassmorphism elements.
